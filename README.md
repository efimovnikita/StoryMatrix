# StoryMatrix 🇮🇹✨

**StoryMatrix** — это интерактивное PWA-приложение для изучения итальянского языка через создание историй. Приложение использует мощь **Mistral AI**, чтобы помогать пользователю строить предложения, расширять словарный запас и моментально исправлять грамматические ошибки.

## 🚀 Основные возможности

-   **Матричный конструктор**: Собирайте предложения из блоков слов (связки, подлежащие, глаголы в разных временах, существительные и т.д.), сгенерированных ИИ под ваш сюжет.
-   **Гибридный ввод**: Переключайтесь между кликами по словам и ручным вводом текста в любой момент.
-   **Умная проверка грамматики**: Получайте детальный разбор ошибок с возможностью «в один клик» применить исправления от нейросети.
-   **Динамический сюжет**: Меняйте «вектор сюжета» прямо во время работы, чтобы получить новую, релевантную лексику.
-   **Публикация в Telegraph**: Сохраняйте свои лучшие истории в сервисе Telegraph с автоматической генерацией креативного заголовка.
-   **PWA и Mobile-First**: Устанавливайте приложение на телефон и работайте с комфортным интерфейсом на любом устройстве.

## 🛠 Технологии

-   **Frontend**: React 18, Vite, TypeScript.
-   **Стилизация**: Tailwind CSS, Lucide Icons.
-   **AI**: Mistral AI API (модели `mistral-large-latest` и `mistral-small-latest`).
-   **Хранение**: LocalStorage (клиентское решение без бэкенда).

## 🏁 Быстрый старт

1.  **Установка зависимостей**:
    ```bash
    npm install
    ```
2.  **Запуск в режиме разработки**:
    ```bash
    npm run dev
    ```
3.  **Использование**: Откройте `http://localhost:5173` в браузере. Для работы вам потребуется **Mistral API Key**.

## 📦 Сборка

Для создания оптимизированной production-версии:
```bash
npm run build
```
Проверить собранное приложение:
```bash
npm run preview
```
